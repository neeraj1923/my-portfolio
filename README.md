Modern, High-Performance Developer Portfolio built with React & Tailwind CSS
This repository contains the source code for my professional portfolio. The site is designed to showcase my experience in Full-Stack Development, AI Integration, and Performance Optimization.  

🚀 Key Features

- Dynamic Project Showcase: An interactive carousel using Swiper.js to highlight diverse projects like AI-powered SaaS and mobile apps.  

- Modern UI/UX: Built with Tailwind CSS for a clean, glassmorphic dark-mode aesthetic.

- Engaging Animations: Powered by Framer Motion for smooth transitions and scroll-reveal effects.

- Fully Responsive: Optimized for all screen sizes, from mobile devices to 4K displays.

- Resume Integration: Integrated PDF download functionality for quick recruiter access.

🛠️ Tech Stack

Frontend: React.js, TypeScript, Tailwind CSS, Vite.

Animations: Framer Motion.

Icons & UI: Lucide React, Radix UI (shadcn/ui).

Carousel: Swiper.js.

Deployment: Vercel.

📂 Project Structure

├── public/               # Static assets (Resume, Favicon, Images)
├── src/
│   ├── components/       # Reusable UI components (Navbar, Hero, Carousel)
│   ├── assets/           # Local images and branding
│   ├── App.jsx           # Main application entry point
│   └── index.css         # Global styles and custom scrollbar configurations
├── tailwind.config.js    # Custom theme and color palette
└── package.json          # Project dependencies

💻 Getting Started
1. Clone the repository
